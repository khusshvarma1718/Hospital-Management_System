# Hospital Management System
A Java Swing/AWT application for managing hospital records with a MySQL backend.

## 🚀 Features
* **Patient Management:** Add, view, and delete patient records.
* **Doctor Database:** Maintain a list of specialized doctors.
* **Database Integration:** Real-time data storage using MySQL and JDBC.

## 🛠️ Tech Stack
* **Language:** Java (JDK 8+)
* **Database:** MySQL 8.0
* **IDE:** Eclipse

## 📋 How to Run
1. **Database:** Run the `database_setup.sql` file in your MySQL Shell to create the schema.
2. **Drivers:** Ensure the `mysql-connector-java.jar` is added to your Eclipse Project Build Path.
3. **Configuration:** Update the `root` password in the Connection string within the Java code.
4. **Run:** Execute `HospitalManagementSystem.java`.
